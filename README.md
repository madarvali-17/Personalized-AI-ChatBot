# Personalized-AI-ChatBot
Madar AI Assistant is a modern web-based chatbot built with Flask and Google Gemini API. It features a vibrant glassmorphism UI using Tailwind CSS, real-time chat interaction, typing indicators, and seamless frontend-backend integration for AI-powered conversations.

**Objectives**

To build an interactive AI chatbot using modern web technologies

To integrate Google Gemini API for natural language responses

To design a visually appealing, responsive chat UI

To demonstrate frontend–backend communication using REST APIs


**System Architecture**

The application follows a client–server architecture:

Frontend: HTML, Tailwind CSS, JavaScript

Backend: Python Flask

AI Engine: Google Gemini 2.5 Flash

Communication: HTTP POST requests (JSON)

**Features**

Real-time AI chat interaction

Glassmorphism and animated UI

Typing indicator animation

Smooth scrolling chat window

Secure API key handling using environment variables

Responsive design for all screen sizes
